VSSStreamSigner Class Reference

Inherits from NSObject

Overview

Wrapper for the functionality for composing/verifying signatures.

This wrapper works with NSInputStream instead of NSData objects.

Lifecycle

– initWithHash:

Designated constructor.

- (instancetype __nonnull)initWithHash:(NSString *__nullable)hash

Parameters

hash

Name of the preferred hash function. In case of nil default hash function will be used (SHA384). One of the following names should be used: kHashNameMD5, kHashNameSHA256, kHashNameSHA384, kHashNameSHA512.

Return Value

Instance of the VSSSigner.
