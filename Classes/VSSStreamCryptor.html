VSSStreamCryptor Class Reference

Inherits from VSSBaseCryptor : NSObject
Declared in VSSStreamCryptor.h

Overview

Class for performing encryption/decryption using NSInputStream as a source of the data and NSOutputStream as a receiver.

Encryption

– encryptDataFromStream:toStream:embedContentInfo:error:

Encrypts the given data using added recepients. Allows to embed info about the recipients so it will be easier to setup decryption.

- (BOOL)encryptDataFromStream:(NSInputStream *__nonnull)source toStream:(NSOutputStream *__nonnull)destination embedContentInfo:(BOOL)embedContentInfo error:(NSError *__nullable *__nullable)error

Parameters

source

Input stream object representing the data which needs to be encrypted.
destination

Output stream object representing the encrypted data location.
embedContentInfo

YES in case when some amount of data with recipients info will be added to the result data, NO - otherwise.
error

NSError pointer to get an object in case of error, nil - otherwise.

Return Value

YES in case when operation completed successfully, NO - otherwise.
