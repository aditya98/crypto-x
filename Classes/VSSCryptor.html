Overview

Class for encryption/decryption functionality.

Encryption

– encryptData:embedContentInfo:error:

Encrypts the given data using added recepients. Allows to embed info about the recipients so it will be easier to setup decryption.

- (NSData *__nullable)encryptData:(NSData *__nonnull)plainData embedContentInfo:(BOOL)embedContentInfo error:(NSError *__nullable *__nullable)error

Parameters

plainData

Data object which needs to be encrypted.
embedContentInfo

YES in case when some amount of data with recipients info will be added to the result data.
error

NSError pointer to get an object in case of error, nil - otherwise.

Return Value

Data object with encrypted data or nil in case of error.

privateKey

Data object containing the private key for decryption (should correspond the public key used for encryption).

error
