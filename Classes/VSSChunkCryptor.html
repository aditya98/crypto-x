Inherits from VSSBaseCryptor : NSObject

Overview

Class for performing encryption/decryption of relatively small parts of data.

Setup

– startEncryptionWithPreferredChunkSize:error:

Performs all necessary setup for the encryption process, using preferred chunk size.

- (size_t)startEncryptionWithPreferredChunkSize:(size_t)chunkSize error:(NSError *__nullable *__nullable)error

Parameters

chunkSize

Preferred size of a single data chunk. In case of 0 - default chunk size will be used (1MiB - 1b for padding).
error

Pointer to NSError object in case of error, nil - otherwise.

Return Value

Actual chunk size which will be used. In case of error 0 will be returned.
